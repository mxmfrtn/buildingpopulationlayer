# Open Building Population Layer (OBPL) - Canada v3

## Disclaimer
The Open Building Population Layer (Canada) is a personal project by Maxim Fortin. It is not an official Government of Canada data source for building footprints or population distribution.

## About
This repository provides the main script used for the **Open Building Population Layer — Canada v3**.

OBPL-Canada v3 is a building-level population approximation for Canada derived from census counts and building footprints. This repository is intentionally minimal and does not contain the full project documentation or data distribution materials.

For methodology, downloads, limitations, and full project details, see:

- **Open Building Population Layer — Canada v3:** https://maximfortin.com/project/obpl-ca-v3/
